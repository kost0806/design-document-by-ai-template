## 2. 인증 테스트

### TC-001: 로그인 성공
- **목적**: 올바른 인증 정보로 로그인 성공 확인
- **입력**: `email: test@example.com`, `password: Test123!`
- **예상 결과**:
  - 상태 코드 200
  - accessToken 반환
- **우선순위**: 🔴 High

### TC-002: 로그인 실패 (잘못된 비밀번호)
- **목적**: 잘못된 비밀번호로 로그인 실패 확인
- **입력**: `email: test@example.com`, `password: wrong`
- **예상 결과**:
  - 상태 코드 401
  - 에러 메시지 반환
- **우선순위**: 🔴 High
