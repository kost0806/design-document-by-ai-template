## 2. 테스트 레벨

### Unit Tests (60%)
- **목표**: 개별 함수/메서드 검증
- **도구**: Jest / Pytest
- **커버리지**: 80% 이상

### Integration Tests (30%)
- **목표**: 컴포넌트 간 통합 검증
- **도구**: Jest / Pytest
- **커버리지**: 70% 이상

### E2E Tests (10%)
- **목표**: 사용자 시나리오 검증
- **도구**: Playwright / Cypress
