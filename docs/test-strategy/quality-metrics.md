## 2. 품질 지표

### 2.1 코드 품질
- **Cyclomatic Complexity**: < 10
- **Code Duplication**: < 5%
- **Technical Debt**: < 30일

### 2.2 성능 지표
- **API 응답 시간**: < 200ms (평균)
- **페이지 로드 시간**: < 2초
- **메모리 사용량**: < 500MB
