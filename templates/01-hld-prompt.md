# HLD (High Level Design) AI 프롬프트 템플릿

---

## 📋 프롬프트 템플릿

```
당신은 소프트웨어 아키텍트입니다. 다음 요구사항을 바탕으로 High Level Design을 작성해주세요.

### 입력: 요구사항 요약
[요구사항 문서의 핵심 내용을 요약하여 붙여넣기]

### 요청사항

1. **시스템 아키텍처**
   - 전체 시스템 구조 다이어그램 (Mermaid 형식)
   - 레이어 아키텍처 정의
   - 권장 기술 스택 (Frontend, Backend, Database, Infrastructure)

2. **주요 컴포넌트**
   - 컴포넌트 식별 및 역할 정의
   - 컴포넌트 간 상호작용 (시퀀스 다이어그램)

3. **인터페이스 설계**
   - 주요 API 엔드포인트 정의
   - 외부 시스템 연동 방식

4. **아키텍처 패턴 추천**
   - [현재 팀 규모], [예상 확장성], [기술 역량]을 고려하여
   - Monolithic vs Microservices vs Serverless 중 추천 및 이유
```

---

## 💡 사용 예시

```
요구사항 요약:
- 사용자 인증 (JWT)
- CRUD 기능
- 실시간 알림 (선택)
- 예상 사용자 10,000명
- 팀 규모: 개발자 3명

아키텍처 추천 및 기술 스택을 제시해주세요.
```
