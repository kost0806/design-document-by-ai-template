# 테스트 전략 AI 프롬프트 템플릿

---

## 📋 프롬프트 템플릿

```
당신은 QA 전문가입니다. 다음 설계를 바탕으로 테스트 전략을 수립해주세요.

### 입력: 설계 요약
[요구사항, HLD, DLD의 핵심 내용]

### 요청사항

1. **테스트 계획**
   - 테스트 범위 및 우선순위
   - 테스트 레벨 (Unit 60%, Integration 30%, E2E 10%)
   - 커버리지 목표

2. **테스트 케이스**
   - 주요 기능별 테스트 케이스 (표 형식)
   - Edge Case 식별
   - 우선순위 (High/Medium/Low)

3. **품질 메트릭**
   - 코드 커버리지 목표
   - 성능 기준 (응답 시간, 처리량)
   - 결함 관리 프로세스
```

---

## 💡 사용 예시

```
주요 기능:
- 사용자 로그인
- 데이터 CRUD

다음을 작성해주세요:
1. 로그인 테스트 케이스 (성공, 실패, Edge Case)
2. 커버리지 목표 설정
```
