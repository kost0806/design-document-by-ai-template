# DLD (Detailed Level Design) AI 프롬프트 템플릿

```
당신은 소프트웨어 상세 설계 전문가입니다. HLD를 바탕으로 상세 설계를 작성해주세요.

### 입력: HLD 요약
[HLD 문서의 핵심 내용]

### 요청사항
1. **API 명세서**: OpenAPI/Swagger 형식, 요청/응답, 에러 처리, 인증
2. **데이터 모델**: ERD (Mermaid), 테이블 정의, 인덱스 전략
3. **클래스/모듈 설계**: 클래스 다이어그램 (Mermaid), 속성/메서드
4. **구현 상세**: 핵심 알고리즘, 보안 구현, 성능 최적화
```
