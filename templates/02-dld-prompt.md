# DLD (Detailed Level Design) AI 프롬프트 템플릿

---

## 📋 프롬프트 템플릿

```
당신은 소프트웨어 상세 설계 전문가입니다. HLD를 바탕으로 상세 설계를 작성해주세요.

### 입력: HLD 요약
[HLD 문서의 핵심 내용]

### 요청사항

1. **API 명세서 (OpenAPI/Swagger 형식)**
   - 엔드포인트별 요청/응답 예시
   - 상태 코드 및 에러 처리
   - 인증 방식

2. **데이터 모델 (ERD)**
   - Mermaid ERD 다이어그램
   - 테이블 정의 (컬럼, 타입, 제약조건)
   - 인덱스 전략

3. **클래스/모듈 설계**
   - 주요 클래스 다이어그램 (Mermaid)
   - 클래스별 속성 및 메서드

4. **구현 상세**
   - 핵심 알고리즘 (의사 코드)
   - 보안 구현 방법
   - 성능 최적화 전략
```

---

## 💡 사용 예시

```
HLD 요약:
- 3-tier 아키텍처
- PostgreSQL 사용
- RESTful API

다음을 작성해주세요:
1. 사용자 인증 API 명세
2. users 테이블 ERD 및 스키마
```
